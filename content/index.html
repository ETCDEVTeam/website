Projects

ETCDEV Team does work on a set of core Ethereum Classic projects

ETC Geth

Fork of original EF Geth, which is a reference implementation of Ethereum Classic protocol now

Emerald SDK

An official desktop wallet, and a flexible SDK to build custom applications on top of Ethereum Classic blockchain

Monetary Policy

ECIP-1017 which adds a cap (5M20) to total issued coins in Ethereum Classic blockchain. The team is working on implementation for both Parity and Geth, does full testing cycle of the new supply model, and examines for possible attacks

Sputnik VM

A new implementation of Ethereum Virtual Machine with main focus on performance and reliability. It can be used as a standalone EVM to help developers to build their Dapps, or embed into other apps
